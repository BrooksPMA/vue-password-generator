# vue-password-generator

Simple password generator made with Vue + tailwind. It has dark and light themes. Enjoy ;)

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
